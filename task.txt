Student Management System (Python)
🎯 Objective

Develop a menu-based Student Management Program using Python fundamentals.

This task will test your understanding of:

Variables

Lists & Dictionaries

if–else conditions

Loops

Functions

Classes

📌 Completion of this task proves real Python understanding.

🟦 TASK REQUIREMENTS (MANDATORY)
🔹 1. Program Menu

Your program must display a menu repeatedly until the user exits:

1. Add Student
2. View All Students
3. Search Student
4. Update Student Marks
5. Delete Student
6. Exit

🔹 2. Student Record Structure

Each student record must store:

Roll Number

Name

Department

Marks

Grade

📌 Use Dictionary for one student and List to store multiple students.

🔹 3. Add Student

Take input from user

Calculate grade using marks:

≥90 → A

≥80 → B

≥70 → C

Else → F

Add student to the list

🔹 4. View All Students

Display all student records

Use a loop to print students

If no students exist, show message:

No students found

🔹 5. Search Student

Search student using Roll Number

If found → display student details

If not found → show error message

🔹 6. Update Student Marks

Ask for Roll Number

Update marks and grade

If student not found, show error

🔹 7. Delete Student

Delete student by Roll Number

Confirm deletion message

Handle case when Roll Number does not exist

🔹 8. Exit Program

Program must stop only when user selects Exit
